# 我的桌面装备 2026

*Published: March 28, 2026*

作为一个每天在电脑前待 12+ 小时的研究生，我对桌面装备有一种执念。好的工具不会直接提升论文质量，但至少能让写论文的过程舒服一点。

这篇记录一下我目前的主力装备，以及一些踩过的坑。

## 主力机：MacBook Pro 14 (M3 Pro)

**配置**：18GB 内存 + 512GB SSD

买它的原因很简单：实验室补贴 + 教育优惠之后价格能接受，而且 M 系列的续航确实香。

**实际使用**：
- 跑 Python 代码没问题，但大规模训练还是得用服务器
- 18GB 内存对于开 20 个 Chrome 标签 + VS Code + 几个终端来说够用
- 续航是真的强，去咖啡店写代码一天不用带充电器

**唯一的遗憾**：应该买 36GB 内存版本的。有时候需要本地跑一个小模型测试，18GB 有点紧张。

## 显示器：LG 27UP850N

**规格**：27 寸 4K IPS，Type-C 65W 供电

之前用的是 24 寸 1080p，换了 4K 之后回不去了。

**为什么选它**：
- 支持 Type-C 一线连，MacBook 接上去同时充电 + 投屏
- 色彩还行，看 paper 和写代码都舒服
- 价格比 Studio Display 便宜太多

**缺点**：
- 60Hz 刷新率，打游戏不太行（反正我也不打）
- 底座占地方，我换了个显示器支架

## 键盘：Keychron K3 Pro

**配置**：红轴矮轴，RGB 背光

用过很多键盘，最后留在了这把。

**优点**：
- 矮轴打字不累，长时间写代码手指不会酸
- 可以同时连 Mac 和服务器（双系统切换方便）
- 声音不大，实验室用不会吵到别人

**缺点**：
- 键帽打油之后有点丑，准备换一套 PBT 的
- RGB 灯光我基本不开，费电且没必要

## 鼠标：Logitech MX Master 3S

这鼠标是去年双十一买的，用了就回不去了。

**最实用的功能**：
- 侧滚轮：看宽表格和代码 diff 神器
- 手势按钮：拇指按下去配合移动，可以快速切换桌面
- 跨设备复制粘贴：Mac 和服务器之间传文字/文件

**缺点**：
- 有点重，不适合FPS游戏（再次确认我不打游戏）
- 价格小贵，但用了两年觉得值

## 耳机：AirPods Pro 2 + Sony WH-1000XM5

**AirPods Pro 2**：
- 通勤和去健身房用
- 降噪够用，通透模式很自然
- 空间音频看剧挺爽

**Sony WH-1000XM5**：
- 在实验室长时间戴，耳罩式不压耳朵
- 降噪比 AirPods 强一档，写代码时开启「专注模式」
- 夏天有点热，开空调解决

**一个小技巧**：在耳机设置里把「连接两台设备」关掉，稳定性会好很多。

## 配件们

**显示器支架**：北弧 E350ez
- 解放桌面空间，可以调整高度和角度
- 安装的时候需要点力气

**桌面充电器**：Anker 737 氮化镓 120W
- 三个口，可以同时充 MacBook、手机和耳机
- 比带三个充电器省事多了

**鼠标垫**：雷蛇重装甲虫 V3
- 大尺寸，可以同时放下鼠标和键盘
- 布面手感好，但不好清洁

**台灯**：小米屏幕挂灯
- 挂在显示器上面，不占桌面空间
- 晚上写代码的时候照亮桌面，不会晃眼

## 服务器访问装备

虽然我们有自己的服务器，但有时候需要远程访问。

**SSH 工具**：iTerm2 + tmux
- iTerm2 比自带终端好用太多
- tmux 保证 SSH 断了会话还在，可以恢复

**远程桌面**：RustDesk
- 开源免费，比 TeamViewer 稳定
- 偶尔需要看可视化结果的时候用

**文件同步**：Syncthing
- 本地和服务器之间同步代码和论文
- 比用 U 盘或者微信传文件靠谱

## 总结

这些装备总价值大概 2 万多，分摊到每天 12 小时的使用时间，我觉得挺值的。

如果让我推荐优先级：
1. **显示器**：4K 大屏是刚需
2. **键盘**：矮轴机械键盘，手指会感谢你的
3. **鼠标**：MX Master 系列，效率神器
4. **耳机**：降噪耳机，专注必备

其他的可以慢慢添置。

最后，装备只是工具，重要的是你在上面做什么。我见过用 3 万块 MacBook Pro 天天刷 B 站的，也见过用 3000 块旧笔记本发顶会的。

找到适合自己的就好。
