# Weekly Reading #1 - 有趣的链接分享

*Published: January 10, 2026*

每周分享一些我阅读到的有趣文章、工具和想法。

## 文章

### [The Rise and Fall of the OLAP Database](https://www.figma.com/blog/the-rise-and-fall-of-the-olap-database/)
Figma 团队分享他们如何构建分析数据库，以及为什么最终放弃了这条路。很好的技术决策案例。

### [Cursor 101](https://cursor101.com/)
一个学习 Cursor 编辑器的资源合集，包含教程、提示词技巧和工作流。

### [The 12 Factor App](https://12factor.net/)
经典的现代应用开发方法论，值得反复阅读。

## 工具

- **[Excalidraw](https://excalidraw.com/)** - 手绘风格的图表工具，非常适合画架构图
- **[Obsidian](https://obsidian.md/)** - 本地优先的笔记软件，支持强大的插件系统
- **[Perplexity](https://perplexity.ai/)** - AI 搜索引擎，比传统搜索更高效

## 想法

> "The best code is no code at all. Every new line of code you willingly bring into the world is code that has to be debugged, code that has to be read and understood, code that has to be supported."
> — Jeff Atwood

这周在重构一个旧项目时深有体会。有时候删除代码比添加代码更有价值。
